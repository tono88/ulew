POS Product Detail

This module allows the seller to view the details associated with the products loaded in a Point Of Sale session. Seller can easily checkout the details of any product by using the click and hold event on any product's' image or any order line on the cart. It also give the ability to the POS user to toggle the Action Pad display in the POS interface.

Salient Features
  • Seller can view the details associated with any product in the POS interface by using the click and hold event on any product's image or any order lines on the screen.
  • Seller can toggle the display of the Action Pad and the Num Pad by using the toggle button. This is helpul in the cases when the Action Pad takes most of the screen's space and the cart is not fully visible.
